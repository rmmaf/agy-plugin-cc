# agy plugin — test / build / CI / tooling layer

Child of [[agy-architecture-overview]]. Pure-Node, zero test-framework deps (`node:test` + `node:assert/strict`). `npm test` = `node --test tests/*.test.mjs`; `npm run build` = `tsc -p tsconfig.app-server.json`; `npm run check-version` = `bump-version --check`.

## Test files (11) — what each locks in
- **`agy-runtime.test.mjs`** — largest; spawns real `agy-companion.mjs` against the fake agy. Locks: setup honesty (`mode:"direct"`, `auth.verified:false` even with creds, sign-in `nextSteps`); task success + `status` shows `completed` with `threadId`; **answerFile persisted + exists** (`hadTextAnswer:true`, `transcriptSource:"transcript.jsonl"`); `transcript_full.jsonl` fallback; empty-run → exit 1 + `diagnostic:"auth-or-incomplete"` (NOT legacy "no transcript entry"); `--model` verbatim; `--resume-last` reuses conversation; **resumed turn keeps same threadId but gets a DISTINCT answerFile** (both persist); review renders `# Antigravity Review`; stop-gate BLOCK after `--enable-review-gate`; **review argv includes `--sandbox`; `--write` argv EXCLUDES `--sandbox` and includes `AGY_SANDBOX_WRITE` flag**; stop-gate ALLOW = empty stdout; `parseStructuredOutput`/`stripCodeFence` (fenced+bare, preserves raw); timeout fast-fail (`AGY_TIMEOUT_MS=300` → `status:1`, message names timeout, returns <10s).
- **`transcript-toolcall.test.mjs`** — THE quirk file (F23/F24). `toolcall` mode (final MODEL entry, empty text, `tool_calls:1`) → **SUCCESS** `status:0`; `empty` mode (no final MODEL entry) → FAILURE `status:1`, `diagnostic:null`, msg "no transcript entry was found"; `full-only` → fallback success; `none` → `diagnostic:"auth-or-incomplete"`.
- **`args-backslash.test.mjs`** — Windows path backslash rule (F01): `C:\Users\me\proj` intact; quoted path w/ spaces; `\` escapes only before `"`/`'`/`\`/space; trailing lone `\` literal; quotes group whitespace.
- **`broker-endpoint.test.mjs`** — `darwin`→`unix:<dir>/broker.sock`; `win32`→`pipe:\\.\pipe\<name>-Antigravity-app-server`; round-trips.
- **`bump-version.test.mjs`** — propagates to all 4 manifests (incl. lockfile top-level + `packages[""]`); tolerates npm v1 lockfile (no `packages`); `--check` reports stale plugin/marketplace versions + exits non-zero.
- **`commands.test.mjs`** — doc-as-spec: review/adversarial prose (review-only, `run_in_background:true`, sizing probe, "treat untracked as reviewable", "Return stdout verbatim", "Do not call BashOutput"); **no `continue.md`** (rescue absorbs continue); rescue locks `subagent_type:"agy:agy-rescue"`, no `Skill(agy:agy-rescue)`, no `context:fork` (#234), model verbatim no `spark` alias, `--effort` unset, thin forwarder; result/cancel `disable-model-invocation:true`; hooks keep SessionStart/End + stop gate; setup uses `antigravity.google` installer + agy Google sign-in.
- **`doc-consistency.test.mjs`** — companion dispatcher exposes `setup/review/adversarial-review/task/status/result/cancel/task-resume-candidate`; every `agy-companion.mjs <sub>` named in docs is implemented; no `openai/` refs; README marketplace = `rmmaf/agy-plugin-cc` (F14 was `openai/...`); `--effort` documented accepted-but-ignored; `agy models` not `codex models`; package name `agy-plugin-cc`.
- **`git.test.mjs`** — dirty→working-tree, clean→branch (content has `Branch Diff`); `{base:"main"}`→branch; no default branch→throws "Pass --base"; tiny→`inline-diff`; untracked nested git dir + broken symlink **skipped not crashed**; large multi-file→`self-collect` (diff withheld, `## Changed Files` present); oversized single file→`self-collect`; untracked text kept in lightweight working-tree context.
- **`process.test.mjs`** — Windows `taskkill /PID /T /F` (`delivered:true`, no POSIX fallback); "process not found" (exit 128) = already stopped, no error.
- **`render.test.mjs`** — `renderReviewResult` degrades on bad shape ("unexpected review shape", "Missing array `findings`", "Raw final message:"); `renderStoredJobResult` prefers pre-rendered, appends `Antigravity session ID` + `Resume in Antigravity: agy --conversation=<id>`.
- **`state.test.mjs`** — `resolveStateDir` tmpdir fallback (16-hex suffix) vs `CLAUDE_PLUGIN_DATA`; atomic writes leave no `.tmp`; **51 jobs → cap 50** + evicted `.json`+`.log` deleted; lock leaves no `.state.lock`; pre-held lock → still succeeds <5s (last-writer-wins), doesn't delete a lock it didn't own.

## `fake-agy-fixture.mjs` — executable spec of the agy CLI contract
`installFakeAgy(binDir)` writes `fake-agy.cjs`, run as `node fake-agy.cjs` via `AGY_BIN=process.execPath` + `AGY_BIN_ARG=<script>` (cross-platform, no shell shim). Recognizes `--version` (→`agy 0.0.0-fake`), `--print`/`-p`/`--prompt`, `--model`, `--conversation`, `-c`/`--continue`, `--sandbox` (no value), `--print-timeout <n>`. **Exits 1 if no `--print`** ("interactive not supported"); **exits 3 if `AGY_STATE_DIR` unset** (proves plugin must export it). Conversation id = explicit / persisted-last / minted UUID `00000000-0000-4000-8000-<counter>`. Answer map: stop-gate prompt→`BLOCK:...` (or `ALLOW:` under `AGY_FAKE_BEHAVIOR=stop-allow`); `Review `→"No material issues..."; resume→"Resumed...Follow-up handled."; else "Handled the requested task.". Writes transcript JSONL (USER echo + final MODEL/DONE/PLANNER_RESPONSE) + `cache/last_conversations.json` + a test state file (`lastRun:{prompt,model,conversation,continue,argv}` for assertions). **Headline: exits 0 with EMPTY stdout** (simulates the `--print` non-TTY bug). Modes: `default` / `full-only` (writes `transcript_full.jsonl` instead) / `none` (no transcript) / `stop-allow`.

## `helpers.mjs`
`makeTempDir` (prefix `codex-plugin-test-` — Codex residue), `writeExecutable` (0o755), `run` (spawnSync; Windows `shell` only for non-absolute commands), `initGitRepo` (`git init -b main` + `commit.gpgsign=false`/`tag.gpgsign=false` so CI commits don't block on signing). `buildEnv`/`readFakeState` for the fake.

## Tooling
- **`bump-version.mjs`** — syncs version across **4 manifests**: `package.json`, `package-lock.json` (top + `packages[""]`, latter optional for v1 lockfiles), `plugins/agy/.claude-plugin/plugin.json`, `.claude-plugin/marketplace.json` (`metadata.version` + `plugins[name=="agy"].version`). `--check [ver]` (defaults to package.json's own version) exits 1 listing each mismatch. `--root` for tests.
- **`tsconfig.app-server.json`** — `noEmit:true` JSDoc TYPE-CHECK (not compile), `allowJs`+`checkJs`, lenient (`strict:false`, `skipLibCheck:true`). Includes only `lib/agy.mjs`, `fs.mjs`, `process.mjs`. **`.generated/app-server-types/` is absent from the repo and not in `.gitignore`** (`.gitignore` ignores `plugins/codex/.generated/` — Codex residue); type-check survives only because `skipLibCheck` + JSDoc-only references.
- **CI `.github/workflows/pull-request-ci.yml`** — `on: pull_request` only; matrix **ubuntu + windows** (Windows IS tested), **Node 22**, `timeout-minutes:10`, `fail-fast:false`; steps `npm ci` → `npm test` → `npm run build`. Does NOT run `check-version` as a step (covered by tests).

## What the tests prove about the real agy runtime (the contract)
Corroborates [[agy-real-runtime-behavior]]: `--print` stdout is empty (answer only in transcript); tool-call/edit turns leave empty final answer but are SUCCESS (current code); `transcript_full.jsonl` is a valid fallback; transcript flush lags exit (settle re-read); no-transcript run = `auth-or-incomplete`; conversation id from `cache/last_conversations.json`; `--write` opts out of `--sandbox`; resume reuses conversation but each turn → distinct answer file; `--model` verbatim, `--effort` no-op; runs must be timeout-bounded; Windows uses named pipes + `taskkill`; state store is per-workspace/atomic/capped-50/lock-degraded. NOTE: the fake **honors** `AGY_STATE_DIR` to stay hermetic, but the **real** agy v1.0.8 IGNORES it — so the tests encode the contract the plugin *wants*, not what live agy obeys.
